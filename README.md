## ✨ 功能特性

- 📋 **80道权威题目** — 涵盖能量获取、信息获取、决策方式、生活方式四大维度
- 🎨 **玻璃态视觉设计** — 现代化毛玻璃风格，深色/浅色主题自由切换
- 📊 **详细结果分析** — 给出 MBTI 类型的同时，提供性格特征、优势、盲点等深度解读
- 📱 **响应式布局** — 完美适配手机、平板、桌面端，随时随地测评
- 🌐 **多语言支持** — 简体中文、繁体中文、英语、日语

## 🚀 快速开始

### 在线体验

直接在浏览器中打开 [index.html](index.html) 即可开始测评。

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/huqingyuan-Python/80-question-MBTI-test.git
cd 80-question-MBTI-test

# 直接打开 index.html（无需安装任何依赖）
open index.html
```

## 📖 使用说明

1. 打开页面后，选择**深色**或**浅色**主题
2. 依次回答 80 道性格测评题目
3. 每道题选择最符合你真实想法的选项
4. 完成全部题目后，点击查看结果
5. 系统将分析你的答案并给出 MBTI 类型及详细解读

> ⚠️ 本测评结果仅供参考，性格是多维度的，请勿将其作为权威判断依据。
