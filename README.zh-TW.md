## ✨ 功能特點

- 📋 **80道權威題目** — 涵蓋能量獲取、資訊獲取、決策方式、生活方式四大維度
- 🎨 **玻璃態視覺設計** — 現代化毛玻璃風格，深色/淺色主題自由切換
- 📊 **詳細結果分析** — 給出 MBTI 類型的同時，提供性格特徵、優勢、盲點等深度解讀
- 📱 **響應式佈局** — 完美適配手機、平板、桌面端，隨時隨地測評
- 🌐 **多語言支援** — 簡體中文、繁體中文、英語、日語

## 🚀 快速開始

```bash
git clone https://github.com/huqingyuan-Python/80-question-MBTI-test.git
cd 80-question-MBTI-test
open index.html
```

## 📖 使用說明

1. 開啟頁面後，選擇**深色**或**淺色**主題
2. 依序回答 80 道性格測評題目
3. 每道題選擇最符合你真實想法的選項
4. 完成全部題目後，點擊查看結果
5. 系統將分析你的答案並給出 MBTI 類型及詳細解讀

> ⚠️ 本測評結果僅供參考，性格是多維度的，請勿將其作為權威判斷依據。
