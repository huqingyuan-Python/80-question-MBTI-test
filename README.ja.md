## ✨ 機能特徴

- 📋 **80題の信頼できる問題** — エネルギー獲得、情報収集、意思決定、生活様式の4つの次元を網羅
- 🎨 **ガラスモーフィズムデザイン** — モダンなすりガラス風デザイン、ダーク/ライトテーマ切り替え対応
- 📊 **詳細な結果分析** — MBTIタイプに加え、性格特徴・長所・弱点などの詳しい解説
- 📱 **レスポンシブレイアウト** — モバイル、タブレット、デスクトップに完全対応
- 🌐 **多言語サポート** — 简体中文、繁体中文、英語、日本語

## 🚀 クイックスタート


## 📖 使い方

1. ページを開き、ダークまたはライトテーマを選択
2. 80題の性格診断問題にすべて回答
3. 自分の真实的想法に最も合った選択肢を選択
4. 送信して結果を確認
5. MBTIタイプと詳細な性格分析结果を取得

> ⚠️ 診断結果は参考用です。性格は多次元的なので、権威ある判断基準として使用しないでください。
