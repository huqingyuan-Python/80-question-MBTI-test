## ✨ Features

- 📋 **80 Authoritative Questions** — Covering four dimensions: energy orientation, information gathering, decision-making, and lifestyle
- 🎨 **Glass Morphism Design** — Modern frosted-glass aesthetic with dark/light theme toggle
- 📊 **Detailed Result Analysis** — MBTI type plus in-depth analysis of personality traits, strengths, and blind spots
- 📱 **Responsive Layout** — Perfect experience on mobile, tablet, and desktop
- 🌐 **Multi-language Support** — Simplified Chinese, Traditional Chinese, English, Japanese

## 🚀 Quick Start

### Online Demo

Open [index.html](index.html) directly in your browser to start.

### Local Run

```bash
git clone https://github.com/huqingyuan-Python/80-question-MBTI-test.git
cd 80-question-MBTI-test
open index.html
```

## 📖 Usage

1. Open the page and choose dark or light theme
2. Answer all 80 personality questions
3. Select the option that best matches your true self
4. Submit and view your MBTI type
5. Get detailed personality analysis and insights

> ⚠️ Results are for reference only. Personality is multi-dimensional; please do not treat them as authoritative judgments.
